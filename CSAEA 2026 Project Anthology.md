# Computer Science & Applied Engineering Academy — Project Anthology

The CSAEA Project Anthology is a living reference of 256 project ideas organized by time commitment, platform, and course level.

These project ideas come from various sources; some are past student projects, some are popular projects found online, others are generated by Mr. Harrell himself.

Find a project idea that you're interested in, then begin research online to find data sheets, tutorials, and other info to help you create something great!

## 1–5 Hours — Micro Projects

---

### Secret Message Encoder
<sub>00</sub>

A command-line program that shifts each character in a plaintext string by a user-chosen integer key and reverses the operation for decryption — the classic Caesar cipher. Requires Python 3.x or Java SE 17+; no external libraries, just standard string manipulation and modular arithmetic over the ASCII range.

### Universal Unit Converter
<sub>01</sub>

A terminal application that converts between units of length, mass, temperature, and volume using hard-coded conversion factors. Requires Python 3.x or Java SE 17+ with the standard library only; no external dependencies or hardware.

### Number Guessing Game
<sub>02</sub>

A loop-driven console program that picks a random number in a range and has the user guess, giving "higher" or "lower" hints each round until they get it. Requires Python 3.x (`random`) or Java SE 17+ (`java.util.Random`); no external libraries or hardware.

### Point-and-Click Calculator
<sub>03</sub>

A windowed calculator app with digit buttons, operator buttons, and a display field that evaluates math expressions as you click. Requires Python 3.x with `tkinter` (included with standard Python) or Java SE 17+ with Swing/JavaFX; no additional hardware.

### Markdown-to-Webpage Converter
<sub>04</sub>

A script that reads a Markdown file — headings, bold, italic, links, code blocks — and spits out a valid HTML webpage. Requires Python 3.x with the `re` module (standard library) or a lightweight library like `markdown2`; no hardware beyond a workstation.

### Programmable LED Light Show
<sub>05</sub>

An Arduino sketch that drives one or more LEDs through programmable blink sequences using `digitalWrite()` and `delay()` or timer interrupts. Requires an Arduino Uno, standard 5 mm LEDs (forward voltage ~2 V), 220 Ω current-limiting resistors (one per LED), a USB cable for upload, and a breadboard with jumper wires. Each LED draws about 15–20 mA.

### Password Strength Meter
<sub>06</sub>

A program that scores a candidate password based on length, character-class diversity, and entropy, returning a tiered strength rating (weak, fair, strong, etc.). Requires Python 3.x (`re`, `math`) or Java SE 17+ (`java.util.regex`); no external libraries or hardware.

### Virtual Dice Roller
<sub>07</sub>

A console or GUI app that simulates rolling any number of dice with any number of sides, totals the results, and optionally shows a frequency histogram. Requires Python 3.x (`random`, optionally `matplotlib` for graphs) or Java SE 17+; no hardware.

### Room Temperature Data Logger
<sub>08</sub>

A Raspberry Pi script that reads a temperature sensor at set intervals and saves timestamped readings to a CSV file. Requires a Raspberry Pi 4/5, a one-wire digital temperature sensor (e.g., DS18B20; 3.3 V logic), a 4.7 kΩ pull-up resistor on the data line, and three jumper wires to GPIO. Software: Raspberry Pi OS, Python 3.x with `w1thermsensor`.

### Palindrome & Anagram Checker
<sub>09</sub>

A program that takes two strings and figures out if either is a palindrome and whether the pair are anagrams of each other by comparing character frequencies. Requires Python 3.x or Java SE 17+; standard library only.

### Network-Wide Ad Blocker
<sub>0A</sub>

A Raspberry Pi set up as a DNS sinkhole that blocks ads and trackers for every device on the network. Requires a Raspberry Pi 4/5, a microSD card (8 GB minimum) with Raspberry Pi OS Lite, an Ethernet cable to the academy subnet, and Pi-hole software installed via its `curl` installer. You will need to reconfigure the router's DHCP settings to point all client DNS to the Pi's static IP.

### Morse Code Buzzer
<sub>0B</sub>

An Arduino sketch that converts text typed into the serial monitor into audible Morse code on a piezo buzzer, and optionally decodes timing back into text. Requires an Arduino Uno, a passive piezo buzzer (3–5 V), a 100 Ω resistor, a breadboard, and jumper wires. Total draw under 30 mA.

### Number Base Converter
<sub>0C</sub>

A console app that converts integers between binary, hexadecimal, octal, and decimal, showing each step of the conversion so you can see how it works. Requires Python 3.x or Java SE 17+; no external libraries.

### Rock-Paper-Scissors Tracker
<sub>0D</sub>

A console game that plays rock-paper-scissors against the computer and keeps running stats — win percentage, loss percentage, and longest streak — across multiple rounds. Requires Python 3.x or Java SE 17+; no external libraries or hardware.

### Mad Libs Story Generator
<sub>0E</sub>

A terminal program that reads a story template with tagged blanks, asks the user for nouns, verbs, and adjectives, then fills them in and prints the ridiculous result. Requires Python 3.x (string formatting, file I/O); no external libraries.

### Reflex Reaction Timer
<sub>0F</sub>

An Arduino circuit that lights an LED at a random delay and measures how many milliseconds it takes you to press a button, showing the result on a serial monitor or a small OLED screen (I²C). Requires an Arduino Uno, one LED, one 220 Ω resistor, a tactile pushbutton, a 10 kΩ pull-down resistor, and a breadboard.

### Knob-Controlled RGB Mixer
<sub>10</sub>

An Arduino sketch that reads three potentiometers (10 kΩ each) and maps their values to the red, green, and blue channels of an RGB LED using PWM, letting you "dial in" any color. Requires an Arduino Uno, an RGB LED, three 220 Ω resistors, three potentiometers, and a breadboard. Total draw under 60 mA.

### Cryptographic Hash Generator
<sub>11</sub>

A command-line tool that computes MD5, SHA-1, and SHA-256 digests of a string or file and prints them side by side so you can compare. Requires Python 3.x (`hashlib`) or Java SE 17+ (`java.security.MessageDigest`); no external libraries.

### Automatic File Sorter
<sub>12</sub>

A script that scans a target folder, moves files into subfolders organized by extension or date, and logs every move it makes. Requires Python 3.x (`os`, `shutil`); no external libraries or hardware.

### Terminal Tic-Tac-Toe
<sub>13</sub>

A two-player Tic-Tac-Toe game played in the terminal with an ASCII grid, input validation, and win/draw detection. Requires Python 3.x or Java SE 17+; no external libraries.

### Joystick Position Grapher
<sub>14</sub>

An Arduino sketch that reads X and Y analog values from a two-axis joystick module and streams them to the Arduino IDE Serial Plotter for real-time graphing. Requires an Arduino Uno, a joystick module (5 V, two analog outputs + button), and a breadboard.

### Network Port Scanner
<sub>15</sub>

A Python script that scans a range of TCP ports on a target IP, reports which ones are open, and optionally grabs service banners. Requires Python 3.x (`socket`); must only be used on the academy's own subnet with instructor permission.

### Build-Your-Own Text Adventure
<sub>16</sub>

A console game engine that reads a JSON file describing rooms, items, and transitions, then lets the player explore the world by typing commands. Requires Python 3.x (`json`); no external libraries.

### Fibonacci & Prime Generator
<sub>17</sub>

A program that generates the Fibonacci sequence to N terms and runs the Sieve of Eratosthenes to N, benchmarking iterative versus recursive approaches. Requires Python 3.x or Java SE 17+; no external libraries.

### JSON File Validator
<sub>18</sub>

A utility that reads a JSON configuration file, checks it for proper formatting, reports missing or unexpected fields, and pretty-prints the result. Requires Python 3.x (`json`, optionally `jsonschema`) or Java SE 17+ with a lightweight JSON library.

### Sewable Blinking LED Badge
<sub>19</sub>

A tiny wearable on an Adafruit Gemma M0 that cycles through color patterns on a sewn-on addressable LED. Requires the Gemma M0, one or two sewable NeoPixels, conductive thread, a coin-cell battery holder (3 V CR2032), and the Arduino IDE with the Adafruit NeoPixel library. Each NeoPixel draws up to ~60 mA at full white.

### Light-Controlled Theremin
<sub>1A</sub>

An Arduino sketch that reads a photoresistor through a voltage divider (10 kΩ) and maps light intensity to a tone frequency on a piezo buzzer — wave your hand over it to play "music." Requires an Arduino Uno, one photoresistor, one 10 kΩ resistor, one piezo buzzer, and a breadboard.

### System Resource Monitor
<sub>1B</sub>

A terminal dashboard that polls your computer's CPU and memory usage every second and draws a live-updating bar chart right in the console. Requires Python 3.x with `psutil` (`pip install`); works on any OS.

### QR Code Maker
<sub>1C</sub>

A script that takes any text or URL and generates a QR code image file you can scan with your phone. Requires Python 3.x with `qrcode` and `Pillow` (`pip install`); no hardware.

### Resistor Color Code Decoder
<sub>1D</sub>

A console or GUI tool that converts between resistance values (in ohms) and the 4- or 5-band color codes printed on resistors, and vice versa. Requires Python 3.x (optionally `tkinter` for a GUI); no hardware.

### Rotary-Knob Countdown Timer
<sub>1E</sub>

An Arduino project where you twist a rotary encoder to set a time, see it count down on an OLED screen (I²C), and hear a buzzer go off when it hits zero. Requires an Arduino Uno, one rotary encoder (e.g., KY-040), one OLED (128×64, I²C), one piezo buzzer, and a breadboard.

### Word Frequency Counter
<sub>1F</sub>

A script that reads a text file, counts how often each word appears (ignoring common stop words), and prints a ranked frequency table — optionally generating a word cloud image. Requires Python 3.x (`collections`, optionally `wordcloud` and `matplotlib`).

### Subnet Address Calculator
<sub>20</sub>

A command-line tool that takes a CIDR notation address (like 192.168.1.0/24) and outputs the network address, broadcast address, host range, and usable host count. Requires Python 3.x (`ipaddress` module, standard library); no external libraries.

### CSV Data Cleanup Tool
<sub>21</sub>

A script that reads a messy CSV file, removes duplicate rows, fills or flags blank cells, normalizes text formatting, and writes a clean version. Requires Python 3.x with `pandas` (`pip install`); no hardware.

### Stack & Queue Playground
<sub>22</sub>

A program that implements stack and queue data structures from scratch — both array-backed and linked-node versions — with an interactive menu for push, pop, enqueue, and dequeue. Requires Python 3.x or Java SE 17+; no external libraries.

### Touch-Toggle Light Switch
<sub>23</sub>

An Arduino circuit that reads a capacitive touch sensor (3–5 V) and toggles an LED on or off with each tap. Requires an Arduino Uno, one touch sensor module, one LED, one 220 Ω resistor, and a breadboard.

### English-to-Braille Translator
<sub>24</sub>

A console program that converts English text to Grade 1 Braille Unicode characters and back again using a lookup table. Requires Python 3.x; no external libraries.

### 3D-Printed Phone Stand
<sub>25</sub>

A parametric phone stand designed in Autodesk Fusion (or Tinkercad) and printed on the academy's 3D printer. Requires Autodesk Fusion with .STL export and a 3D printer with PLA filament; no electronics. You turn in the CAD file and the printed object.

### Analog Sine Wave Generator
<sub>26</sub>

An Arduino sketch that computes sine-wave samples and writes them to an external digital-to-analog converter (DAC, I²C, 12-bit) to produce a smooth analog waveform you can verify on a speaker or oscilloscope. Requires an Arduino Uno, one DAC breakout (e.g., MCP4725), and a breadboard. Output current under 25 mA.

### Keyword-Matching Chatbot
<sub>27</sub>

A console chatbot that matches your input against keyword and pattern rules stored in a JSON file, returns scripted responses, and tracks basic conversation state. Requires Python 3.x (`re`, `json`); no external libraries.

### Magnetic Field Alarm
<sub>28</sub>

An Arduino project that reads a Hall-effect sensor (5 V) and lights an LED or sounds a buzzer when a magnet gets too close. Requires an Arduino Uno, one Hall-effect sensor, one 10 kΩ pull-up resistor, one LED, one 220 Ω resistor, and a breadboard.

### Pomodoro Study Timer
<sub>29</sub>

A terminal-based Pomodoro timer that cycles through 25-minute work sessions and 5-minute breaks, beeping or flashing at each transition and logging completed sessions to a file. Requires Python 3.x (`time`, `os`); no hardware.

### Network Ping Sweep Tool
<sub>2A</sub>

A script that pings every address in a subnet range and reports which hosts are alive, how fast they respond, and (optionally) their hostnames. Requires Python 3.x (`subprocess` or `scapy`); must run on the academy subnet with instructor permission.

### Matrix Digital Rain
<sub>2B</sub>

A terminal app that simulates the famous falling-character "digital rain" effect using ANSI escape codes, with adjustable speed and character sets. Requires Python 3.x (`curses` on Linux or ANSI via `sys`); no external libraries on Linux.

### Logic Gate Simulator
<sub>2C</sub>

A program that models AND, OR, NOT, NAND, NOR, XOR, and XNOR gates, accepts a gate-network description or truth-table input, and evaluates every possible output combination. Requires Python 3.x or Java SE 17+; no external libraries.

### Copper-Wire Electromagnet
<sub>2D</sub>

A hands-on demo where you wind copper wire around an iron bolt to build an electromagnet, power it with a battery (9 V or 4×AA), and use an Arduino with a Hall-effect sensor to measure field strength versus number of coil turns. Requires an Arduino Uno, copper wire (22–26 AWG from the academy supply), an iron bolt, a battery holder, a Hall-effect sensor, and a breadboard.

### Thermal Receipt Printer
<sub>2E</sub>

A Raspberry Pi script that formats and prints little receipts — student schedules, event tickets, quiz scores — on a small thermal printer. Requires a Raspberry Pi 4/5, a serial thermal printer (5–9 V, ~1.5 A peak, needs its own power supply), and Python 3.x with `python-escpos` or `serial`.

### Bitwise Operations Visualizer
<sub>2F</sub>

A GUI or console tool that walks through AND, OR, XOR, NOT, and shift operations step by step on user-supplied integers, showing each operand and result in binary, hex, and decimal simultaneously. Requires Python 3.x (optionally `tkinter`) or Java SE 17+; no hardware.

### Wearable Step Counter

A pedometer built on an Adafruit Gemma M0 that reads an accelerometer (I²C, 3.3 V) to count steps, showing the total via NeoPixel color thresholds or a tiny OLED. Requires a Gemma M0, one accelerometer breakout, NeoPixels or a small OLED, conductive thread, and a LiPo battery (3.7 V, 150+ mAh).

### Joystick-Aimed Servo Turret
